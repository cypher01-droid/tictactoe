# Tic-Tac-Toe Deluxe

An **unbeatable** Tic-Tac-Toe game with:

- **Dark / Light theme toggle**
- **Minimax AI** (you can only draw!)
- **Win animation**
- **Score tracking**
- **Thinking progress bar**
- **Thread-safe, no GUI freeze**

---

## How to Play

- **Fire** = You (Player X)  
- **Water** = AI (Player O)  
- Click any empty cell to place your mark.  
- The AI responds instantly with perfect play.
